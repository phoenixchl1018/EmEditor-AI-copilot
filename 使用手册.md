## 简介

**EmEditor AI助手插件**是一款为EmEditor文本编辑器开发的智能AI对话插件。它允许用户在编辑文本时，随时选中内容并与多种AI平台进行交互，实现文本总结、解释、翻译、代码审查等多种功能。

支持的AI平台包括：
- **DeepSeek** - 深度求索大模型
- **豆包 (火山引擎)** - 字节跳动AI服务
- **Kimi** - 月之暗面大模型
- **MiniMax** - MiniMax大模型
- **通义千问** - 阿里云大模型
- **OpenAI GPT** - ChatGPT API
- **Google Gemini** - Google大模型
- **自定义** - 支持OpenAI兼容格式的其他API

---

## 功能特性

### 核心功能
1. **智能选中文本识别** - 选中文本后自动弹出AI助手窗口
2. **多种AI功能**
   - 总结 - 对选中文本进行智能总结
   - 解释 - 解释概念或代码含义
   - 翻译 - 支持多语言翻译
   - 对话 - 自由对话模式
   - 代码审查 - 审查代码质量
   - 语法检查 - 检查文本语法
   - 改写 - 优化文本表达

3. **多平台AI支持** - 同时配置多个AI平台，随时切换
4. **历史记录** - 保存对话历史，支持搜索和导出
5. **自定义界面** - 可调节窗口大小、字体大小、主题
6. **迷你输入框** - 简洁的输入界面，快速发送消息

### 界面特性
- 悬浮窗口可自由调节大小
- 字体大小可调（8-24pt）
- 支持浅色/深色主题
- 类似EmEditor的界面风格
- 状态栏实时显示操作状态

## 系统要求

### 最低配置
- **操作系统**: Windows 7 SP1 或更高版本
- **EmEditor版本**: v20.0 或更高版本
- **内存**: 2 GB RAM
- **网络**: 需要互联网连接（用于AI API调用）

### 推荐配置
- **操作系统**: Windows 10/11 64位
- **EmEditor版本**: v23.0 或更高版本
- **内存**: 4 GB RAM 或更高
- **网络**: 稳定的宽带连接

## 安装方法

### 方法一：自动安装（推荐）

1. 下载插件安装包 `EmEditorAIPlugin_Setup.exe`
2. 关闭EmEditor编辑器
3. 运行安装程序，按提示完成安装
4. 启动EmEditor，插件将自动加载

### 方法二：手动安装

1. 下载插件文件 `EmEditorAIPlugin.dll`
2. 关闭EmEditor编辑器
3. 将DLL文件复制到EmEditor插件目录：
   ```
   C:\Program Files\EmEditor\PlugIns\
   ```
   或
   ```
   %USERPROFILE%\AppData\Roaming\Emurasoft\EmEditor\PlugIns\
   ```
4. 启动EmEditor
5. 在菜单栏选择 **工具(T) → 插件(P) → AI助手** 启用插件

### 方法三：从源代码编译

1. 安装 Visual Studio 2022 或更高版本
2. 安装 Windows SDK 10.0 或更高版本
3. 克隆或下载源代码
4. 打开 `EmEditorAIPlugin.sln` 解决方案文件
5. 选择 Release 配置，编译项目
6. 将生成的 `EmEditorAIPlugin.dll` 复制到插件目录

## 配置AI平台

### 获取API密钥

#### DeepSeek
1. 访问 https://platform.deepseek.com/
2. 注册并登录账号
3. 进入"API Keys"页面
4. 创建新的API密钥
5. 复制密钥备用

#### 豆包 (火山引擎)
1. 访问 https://console.volcengine.com/
2. 注册火山引擎账号
3. 进入"方舟"控制台
4. 创建推理接入点
5. 获取API Key和模型ID

#### Kimi (Moonshot)
1. 访问 https://platform.moonshot.cn/
2. 注册开发者账号
3. 创建API密钥
4. 复制密钥备用

#### MiniMax
1. 访问 https://api.minimax.chat/
2. 注册开发者账号
3. 获取API密钥和Group ID

#### 通义千问
1. 访问 https://dashscope.aliyun.com/
2. 使用阿里云账号登录
3. 开通DashScope服务
4. 创建API-KEY

#### OpenAI GPT
1. 访问 https://platform.openai.com/
2. 注册OpenAI账号
3. 进入API Keys页面
4. 创建新的API密钥

#### Google Gemini
1. 访问 https://ai.google.dev/
2. 使用Google账号登录
3. 获取API密钥

### 配置插件

1. 点击AI助手窗口的 **设置** 按钮
2. 选择要配置的AI平台
3. 填写以下信息：
   - **API地址** - 平台的API端点URL
   - **API密钥** - 从平台获取的密钥
   - **模型名称** - 使用的AI模型
4. 点击 **测试连接** 验证配置
5. 点击 **确定** 保存设置

### 预设配置参考

| 平台 | API地址 | 默认模型 |
|------|---------|----------|
| DeepSeek | `https://api.deepseek.com/v1/chat/completions` | `deepseek-chat` |
| 豆包 | `https://ark.cn-beijing.volces.com/api/v3/chat/completions` | `doubao-1.5-pro-32k` |
| Kimi | `https://api.moonshot.cn/v1/chat/completions` | `moonshot-v1-8k` |
| MiniMax | `https://api.minimax.chat/v1/text/chatcompletion_v2` | `abab6.5s-chat` |
| 通义千问 | `https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation` | `qwen-turbo` |
| OpenAI | `https://api.openai.com/v1/chat/completions` | `gpt-3.5-turbo` |
| Gemini | `https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent` | `gemini-pro` |

## 使用方法

### 基本操作

#### 1. 打开AI助手
- **菜单方式**: 工具(T) → 插件(P) → AI助手
- **快捷键**: 可在EmEditor中自定义快捷键
- **自动弹出**: 选中文本时自动显示（需在设置中开启）

#### 2. 使用AI功能

**总结功能**
1. 在编辑器中选中要总结的文本
2. 点击AI助手窗口的 **总结** 按钮
3. 等待AI生成总结内容

**解释功能**
1. 选中需要解释的文本或代码
2. 点击 **解释** 按钮
3. 查看AI的详细解释

**翻译功能**
1. 选中要翻译的文本
2. 点击 **翻译** 按钮
3. 在设置中选择目标语言
4. 查看翻译结果

**自由对话**
1. 在输入框中输入问题
2. 点击 **发送** 按钮或按 Enter
3. 查看AI回复

#### 3. 调整界面

**调节窗口大小**
- 拖动窗口边缘即可自由调整大小
- 窗口大小会自动保存

**调节字体大小**
- 使用底部的字体大小滑块
- 或点击设置进行精确调整

**切换主题**
- 在设置中选择浅色或深色主题

### 高级功能

#### 历史记录管理
1. 点击 **历史** 按钮打开历史记录窗口
2. 查看所有对话记录
3. 点击某条记录查看详情
4. 点击 **使用此消息** 将内容复制到输入框
5. 点击 **导出** 将历史记录保存为文本文件

#### 多平台切换
1. 在AI助手窗口顶部的下拉框中选择AI平台
2. 切换后立即生效，新消息将使用选中的平台

#### 代码审查
1. 选中代码片段
2. 在输入框中输入 `/review` 或点击代码审查功能
3. 查看AI的代码审查意见

## 设置说明

### 常规设置

| 选项 | 说明 |
|------|------|
| 选中文本时自动显示AI助手 | 选中文字后自动弹出AI助手窗口 |
| 记住窗口大小和位置 | 下次打开时恢复上次的大小和位置 |
| 窗口置顶 | AI助手窗口始终在最前面 |
| 保存对话历史 | 自动保存对话记录到本地 |
| 历史记录保留天数 | 自动清理超过指定天数的历史记录 |

### 字体设置

| 选项 | 说明 |
|------|------|
| 字体大小 | 调整输入和输出区域的字体大小（8-24pt） |
| 字体名称 | 选择显示字体（推荐微软雅黑） |
| 粗体 | 使用粗体显示 |
| 主题 | 选择浅色或深色主题 |

### AI参数设置

| 选项 | 说明 | 建议值 |
|------|------|--------|
| 最大Token数 | AI回复的最大长度 | 2048 |
| Temperature | 创造性程度，越低越确定 | 0.7 |
| 默认翻译语言 | 翻译功能的默认目标语言 | 简体中文 |

### AI平台设置

| 选项 | 说明 |
|------|------|
| AI平台 | 选择要配置的AI平台 |
| API地址 | 平台的API端点URL |
| API密钥 | 从平台获取的访问密钥 |
| 模型名称 | 使用的AI模型名称 |
| 测试连接 | 验证配置是否正确 |

## 常见问题

### Q1: 插件无法加载怎么办？

**A:**
1. 确认EmEditor版本是否满足要求（v20.0+）
2. 检查DLL文件是否在正确的插件目录
3. 确认系统是否安装了必要的运行库（Visual C++ Redistributable）
4. 尝试以管理员身份运行EmEditor

### Q2: 连接AI平台失败怎么办？

**A:**
1. 检查网络连接是否正常
2. 确认API密钥是否正确
3. 检查API地址是否填写正确
4. 确认API密钥是否有足够的额度
5. 点击"测试连接"查看具体错误信息

### Q3: 如何更换AI平台？

**A:**
1. 在AI助手窗口顶部的下拉框中选择其他平台
2. 或在设置中配置新的AI平台并设为默认

### Q4: 历史记录保存在哪里？

**A:**
历史记录保存在以下位置：
```
%APPDATA%\EmEditorAIPlugin\AIChatHistory.json
```

### Q5: 如何备份配置？

**A:**
配置文件保存在：
```
%APPDATA%\EmEditorAIPlugin\EmEditorAIPlugin.ini
```
复制此文件即可备份配置。

### Q6: 支持代理服务器吗？

**A:**
插件使用系统代理设置。如需配置代理，请在Windows系统设置中配置。

### Q7: 如何卸载插件？

**A:**
1. 关闭EmEditor
2. 删除插件目录中的 `EmEditorAIPlugin.dll`
3. 删除配置目录 `%APPDATA%\EmEditorAIPlugin\`（可选）

### Q8: 为什么AI回复很慢？

**A:**
1. 检查网络连接速度
2. 尝试切换其他AI平台
3. 减少请求的文本长度
4. 检查AI平台的API响应时间
