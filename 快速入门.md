# EmEditor AI助手插件 - 快速入门指南

## 5分钟快速上手

### 第一步：获取插件

#### 方式A：下载预编译版本（推荐）
1. 从发布页面下载 `EmEditorAIPlugin.dll`
2. 跳转到第三步

#### 方式B：从源代码编译
1. 确保已安装 Visual Studio 2022
2. 双击运行 `build.bat`
3. 选择 Release 配置和 x64 平台
4. 等待编译完成

### 第二步：安装插件

1. **关闭 EmEditor**（重要！）
2. 复制 `EmEditorAIPlugin.dll` 到插件目录：
   ```
   C:\Program Files\EmEditor\PlugIns\
   ```
3. 启动 EmEditor

### 第三步：配置AI平台

1. 在 EmEditor 中选择 **工具 → 插件 → AI助手**
2. 点击 AI助手窗口的 **设置** 按钮
3. 选择你想使用的AI平台（如 DeepSeek）
4. 填写API信息：

| 字段 | 示例值 |
|------|--------|
| API地址 | `https://api.deepseek.com/v1/chat/completions` |
| API密钥 | `sk-xxxxxxxxxxxxxxxxxxxxxxxx` |
| 模型名称 | `deepseek-chat` |

5. 点击 **测试连接** 验证
6. 点击 **确定** 保存

### 第四步：开始使用

#### 使用总结功能
1. 在编辑器中选中一段文本
2. 点击 **总结** 按钮
3. 查看AI生成的总结

#### 使用翻译功能
1. 选中要翻译的文本
2. 点击 **翻译** 按钮
3. 在设置中选择目标语言
4. 查看翻译结果

#### 自由对话
1. 在底部输入框中输入问题
2. 按 **Enter** 或点击 **发送**
3. 查看AI回复

## 获取API密钥

### DeepSeek（推荐，国内可用）
1. 访问 https://platform.deepseek.com/
2. 注册账号
3. 进入 "API Keys" 页面
4. 创建新密钥
5. 复制密钥到插件设置

### 豆包（火山引擎）
1. 访问 https://console.volcengine.com/
2. 注册火山引擎账号
3. 进入 "方舟" 控制台
4. 创建推理接入点
5. 获取API Key

### Kimi
1. 访问 https://platform.moonshot.cn/
2. 注册开发者账号
3. 创建API密钥

## 故障排除

### 插件没有显示在菜单中
- 确认DLL文件放在正确的插件目录
- 重启EmEditor
- 检查EmEditor版本是否支持（需要v20.0+）

### 连接测试失败
- 检查网络连接
- 确认API密钥正确
- 检查API地址是否正确
- 确认API密钥有余额

### AI回复很慢
- 检查网络速度
- 尝试切换其他AI平台
- 减少选中的文本长度
